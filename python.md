## Virtual environment
Install the virtualenv package:
```
pip install virtualenv
```
Create the virtual environment:
```
virtualenv mypython
```
Activate the virtual environment:
```
source mypython/bin/activate
```
Deactivate the virtual environment:
```
deactivate
```
